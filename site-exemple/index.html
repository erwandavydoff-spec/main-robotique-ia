Main Robotique

Une main articulée pensée pour traduire des informations (capteurs de flexion et/ou reconnaissance de gestes) en mouvements via des servomoteurs.

En bref

D’un côté, l’Arduino lit les capteurs. De l’autre, une partie “IA” (option) peut reconnaître un geste via caméra.

Arduino + servos Capteurs de flexion Mediapipe + TensorFlow (option)
